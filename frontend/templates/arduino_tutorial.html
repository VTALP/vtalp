Introduction

Arduino is a beginner-friendly microcontroller platform. It connects software instructions to physical devices such as LEDs, buttons, sensors, and motors.

Device Parts and Functions

  • Digital pins: Read or write HIGH and LOW states.
  • Analog pins: Read changing sensor values.
  • Microcontroller: Runs the uploaded program logic.
  • Power indicator: Shows that the board is active.

Lesson Sections

What is Arduino?

Arduino is a microcontroller board used to build embedded systems and IoT prototypes.

Digital Pins

Digital pins work with HIGH and LOW signals for LEDs, buttons, and simple devices.

Analog Pins

Analog pins read variable sensor values such as temperature and light intensity.

LED Control

An LED demonstrates digital output control.

Sensor Input

Sensors convert physical conditions into values the program can use.

Servo Motor Control

A servo moves to an angle commanded by the program.

Step-by-Step Usage Guide

  1. Review the digital and analog pin areas.
  2. Turn the LED on and observe the pin state.
  3. Press the virtual button and read the input state.
  4. Adjust temperature and light sensor sliders.
  5. Rotate the servo and observe the angle display.

Common Mistakes

  • Confusing digital pins with analog pins.
  • Expecting sensor values to behave like ON/OFF signals.
  • Forcing a servo beyond its 0 to 180 degree range.

Safety and Best Practice Tips

  • Check pin purpose before connecting components in real circuits.
  • Use resistors with real LEDs.
  • Power motors and servos carefully in physical builds.
