Device Parts and Functions

  • Display: Shows the measured value and unit.
  • Rotary selector: Chooses voltage, resistance, or continuity mode.
  • COM port: Holds the black probe.
  • V/Ohm port: Holds the red probe for beginner measurements.

Lesson Sections

What is a DMM?

A DMM measures electrical values and helps diagnose circuits.

Voltage Measurement

Use DC Voltage mode on powered DC sources such as batteries.

Resistance Measurement

Use resistance mode only when the circuit is unpowered.

Continuity Testing

Continuity mode checks whether a complete path exists.

Probe Placement

Place the black probe on the reference point and the red probe on the test point.

Step-by-Step Usage Guide

  1. Connect the black probe to COM and the red probe to V/Ohm.
  2. Select the correct mode for the measurement.
  3. Choose the correct sample or circuit point.
  4. Touch both probes to different test points.
  5. Read the value and confirm the unit.

Polarity and Negative Readings

DC voltage has direction. When the red probe touches the positive terminal and the black probe touches the negative terminal, the meter shows a positive voltage such as 9.00 V.

If the probes are reversed, the measurement is still valid. The meter shows a negative value such as -9.00 V to show that the polarity is opposite.

Common Mistakes

  • Using resistance mode on a powered battery.
  • Placing both probes on the same point.
  • Using continuity mode when voltage should be measured.
  • Treating a negative DC voltage as an error instead of a reversed-polarity reading.
  • Leaving the red probe in the 10A port after a current measurement.

Safety and Best Practice Tips

  • Start with the correct mode before touching the probes to a circuit.
  • Measure resistance and continuity only on unpowered circuits.
  • Stop and review if the meter shows overload or error.
  • Use voltage ranges only when measuring across a source or component.
  • Use current ranges only when the meter is placed in series with the circuit.
  • Keep the black probe in COM for normal beginner measurements.
